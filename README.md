# Cars

Сервис для бронирования автомобилей.  
На главной странице представлен каталог марок и карточки автомобилей с ценами в зависимости от срока аренды и кнопкой **Забронировать**.

## Демо
Работающая версия доступна на GitHub Pages:  
[marselchik73.github.io/cars](https://marselchik73.github.io/cars/)

## О проекте

Проект создан мной в рамках интенсива по Frontend-разработке (Айтилогия), под руководством наставника. Планирую использовать его для тестирования.  
Бэкенд реализован как тестовый, поэтому все запросы обрабатываются без реального бронирования.
