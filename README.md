# Cars

Сервис для бронирования автомобилей.  
На главной странице представлен каталог марок и карточки автомобилей с ценами в зависимости от срока аренды и кнопкой **Забронировать**.

Проект создан в демонстрационных целях. Планируется использовать его для тестирования.  
Бэкенд реализован как тестовый, поэтому все запросы обрабатываются без реального бронирования.
